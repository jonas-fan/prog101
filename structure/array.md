# Array

A sequence of data elements stored in contiguous memory locations.

|  | Access | Search | Insert | Delete |
|---|---|---|---|---|
| Array | O(1) | O(n) | - | - |
| Dynamic Array | O(1) | O(n) | O(n) | O(n) |

- (Static) Array: Fixed size. Cannot be resized after initialization.
- Dynamic Array: Usually doubles its capacity (reallocation) when it reaches its limit.
