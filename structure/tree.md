# Tree

A hierarchical structure where each node references its child nodes.

|                    | Access | Search   | Insert   | Delete   |
|--------------------|--------|----------|----------|----------|
| Binary search tree | -      | O(n)     | O(n)     | O(n)     |
| Red-Black tree     | -      | O(log n) | O(log n) | O(log n) |

> [!NOTE]
>
> From abstract data type perspective, a tree does not support access operation.
